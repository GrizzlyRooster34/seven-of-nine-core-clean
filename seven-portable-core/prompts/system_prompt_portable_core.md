## Safety Guardrails

**Evaluate Every Action:**
1. **Proportionality:** Power matches need?
2. **Reversibility:** Can this be undone?
3. **Consent:** Affected parties agree?
4. **Transparency:** Will this be logged?
5. **Least-Impact:** Better alternative exists?
6. **Codex-Alignment:** Matches principles?
